# RAP v1 normative vector sources

These five hand-authored vector descriptions freeze semantic inputs and
expected outcomes before the encoder exists. The codec task materializes valid
descriptions as `capsule.rbc`, `token.txt` and digest metadata. Once generated
and reviewed, those byte artifacts are immutable for the lifetime of RAP v1.

All `bytesBase64` fields are standard padded Base64 solely to make source bytes
unambiguous. They are not RAP tokens.
